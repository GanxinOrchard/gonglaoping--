## 🔧 技術實現

### HTML 結構
```html
<div class="cart-page">
  <div class="cart-container">
    <!-- 標題區 -->
    <div class="cart-header">
      <h1>我的購物車 (N 件商品)</h1>
    </div>
    
    <!-- 主要內容 -->
    <div class="cart-content">
      <!-- 商品列表 -->
      <div class="cart-items-section">
        <div class="cart-item">...</div>
      </div>
      
      <!-- 結帳摘要 -->
      <div class="checkout-summary">...</div>
    </div>
  </div>
</div>
```

### JavaScript 功能
```javascript
// 主要函數
renderCart()           // 渲染購物車
updatePriceSummary()   // 更新價格
updateQuantity()       // 更新數量
removeFromCart()       // 刪除商品
applyCoupon()          // 套用優惠券
proceedToCheckout()    // 前往結帳

// 自動執行
document.addEventListener('DOMContentLoaded', renderCart);
```

### CSS 響應式
```css
/* 桌面版 (> 1024px) */
.cart-content { grid-template-columns: 1fr 400px; }
.checkout-summary { position: sticky; top: 20px; }

/* 平板版 (≤ 1024px) */
.cart-content { grid-template-columns: 1fr; }
.checkout-summary { position: static; }

/* 手機版 (≤ 768px) */
.cart-item { grid-template-columns: 80px 1fr; }
.cart-item-image { width: 80px; height: 80px; }
```
