### 實際範例

```

    ↓
【最新消息管理】工作表（後台）
    ↓
images/pages/news/（圖片）
    ├── cover/（頁面封面）
    └── articles/（文章圖片）
        ├── 2025-01/（按月分類）
        └── 2025-02/
    ↓
自動化：輸入文案 → 點選「發布新文章」→ 自動上線
```

**商品頁**：
```

    ↓
【商品管理】工作表（後台）
    ↓

    └── items/
        ├── ponkan/（椪柑）

        ├── murcott/（茂谷柑）
```

## 🤖 核心自動化功能

### 1. 自動發布文章 ⭐

**操作流程**：
```
1. 開啟【最新消息管理】工作表
2. 新增一行：
   - 標題：2025春季椪柑上市
   - 分類：新品上市
   - 內容：今年春季椪柑品質...
   
3. 點選：📰 發布新文章

4. 系統自動：
   ✅ 生成文章ID (NEWS1234567890)
   ✅ 設定圖片路徑 (images/pages/news/articles/2025-01/...)
   ✅ 更新發布日期
   ✅ 匯出 JSON
   ✅ 同步到前端

5. 完成！文章自動上線
```

### 2. 自動上架商品 ⭐

**操作流程**：
```
1. 開啟【商品管理】工作表
2. 新增一行：
   - 商品名稱：特級椪柑 5斤裝
   - 分類：椪柑
   - 價格：600
   - 庫存：100
   
3. 點選：🛍️  上架新商品

4. 系統自動：
   ✅ 生成商品ID (PROD1234567890)
   ✅ 設定圖片路徑 (images/pages/products/items/ponkan/...)
   ✅ 更新時間
   ✅ 匯出 JSON
   ✅ 同步到前端

5. 完成！商品自動上架
```
