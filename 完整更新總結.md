## 🎬 動畫效果

### 點擊效果
```css
.mobile-menu-toggle:active::before {
    transform: scale(0.95);
}
```
- 按下時橘子縮小至 95%
- 平滑彈性回彈

### 開啟動畫
```css
/* 第一條線 */
transform: translateY(8.5px) rotate(45deg);

/* 第二條線 */
opacity: 0;

/* 第三條線 */
transform: translateY(-8.5px) rotate(-45deg);

/* 小橘子和葉子 */
opacity: 0;
```
- 三條線形成 X 形狀
- 小裝飾淡出
- 過渡時間: 0.3s
