# 4个页面统一修复方案

## 问题
所有4个页面都同时有：
1. 手写的简化header（LOGO路径错误：`images/柑心商標.png`）
2. Template-loader容器（`<div id="header-container"></div>`）
3. 两者冲突导致显示混乱

## 修复方案
删除所有手写header，只保留template-loader容器

## 每个页面需要修改的内容：

### 结构应该是：
```html
<body>
    <!-- Header由template-loader加载 -->
    <div id="header-container"></div>
    <div id="mobile-menu-container"></div>
    
    <!-- 面包屑 -->
    <nav class="breadcrumb-nav" style="margin-top: 120px;">
        ...
    </nav>
    
    <!-- 主要内容 -->
    <main>
        ...
    </main>
    
    <!-- Footer由template-loader加载 -->
    <div id="footer-container"></div>
    
    <!-- JavaScript -->
    <script src="./js/template-loader.js"></script>
    <script src="./js/mobile-menu-simple.js" defer></script>
    <script src="./js/dropdown-menu.js" defer></script>
    <script src="./js/products.js"></script>
    <script src="./js/cart.js"></script>
    <script src="./js/main.js" defer></script>
</body>
```
